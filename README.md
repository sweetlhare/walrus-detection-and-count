# walrus-detection-and-count
Решение кейса "ИИ на страже популяции ненецких моржей" для хакатона в Хабаровске в рамках Цифрового Прорыва: сезон ИИ.

Алгоритм построен на Mask R-CNN из библиотеки mmdetection.
